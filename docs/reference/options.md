!!! note "Research simulation, not live trading"

    transaction costs. It is a backtesting / research tool — not a live broker,
    order-management system, or exchange connection. See the
    [disclaimer](https://github.com/bravant-oss/jaxfolio/blob/main/DISCLAIMER.md).
